FILLING ARRAYS
FROM USER
// Manual — verbose but clear int age[5]; scanf("%d", &age[0]); scanf("%d", &age[1]); // ...up to [4] // Cleaner — loop version for(int i=0; i<5; i++) { scanf("%d", &age[i]); } for(int i=0; i<5; i++) { printf("%d\n", age[i]); }
KEY INSIGHT
You wrote the same program two ways — manual then loop. That's exactly how engineers think: make it work, then make it clean.

PROGRAM 4 · MARKS CALCULATOR 🔥
5 / 7
REAL-WORLD
CALCULATION
int marks[5], sum=0; float average; for(int i=0; i<5; i++) { scanf("%d", &marks[i]); sum = sum + marks[i]; } average = sum / 5.0; printf("sum=%d\n", sum); printf("avg=%.2f\n", average);
→ sum/5.0 not sum/5 — avoids integer division
→ %.2f prints 2 decimal places cleanly
→ accumulator pattern: sum inside the loop
⚡ This exact pattern — collect N readings, sum, divide — is how microcontrollers average sensor noise in real firmware.
