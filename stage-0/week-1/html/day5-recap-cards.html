MILESTONE PROJECT
// 4 functions declared up top float add(float a, float b); float subtract(float a, float b); float multiply(float a, float b); float divide(float a, float b); while(1) { // infinite loop printf("=== MENU ===\n"); scanf("%d", &choice); if(choice==5) break; switch(choice) { case 1: add(a,b); break; // ...all 4 ops } }
→ Function declarations before main()
→ while(1) infinite loop — like Arduino!
→ Division by zero check — professional!
→ break exits loop when user picks 5
⚡ This structure — menu loop + switch + functions — is the exact pattern used in embedded command-line interfaces for testing firmware.
