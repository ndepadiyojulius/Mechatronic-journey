PROGRAM 4 · SWITCH STATEMENT
NUMBER TO
DAY OF WEEK
scanf("%d", &number); switch (number) { case 1: printf("sunday"); break; case 2: printf("monday"); break; case 3: printf("wednesday"); break; // ... up to 7 default: printf("invalid"); }
⚡ In PLC programming, a switch like this is how machines choose between operating modes — manual, auto, emergency, idle.
Don't forget break; — or every case runs! 👇

PROGRAM 5 · WHILE LOOP
LOOP UNTIL
IT STOPS
int count; while (count < 6) { printf("julius\n"); printf("count=%d\n", count); count = count + 1; }
→ Prints "julius" 6 times (count 0 to 5)
→ count = count + 1 same as count++
→ while(1) = infinite loop — every Arduino runs one
💡 Honest note: count was uninitialized — this can cause bugs. Always set int count = 0; at start.
