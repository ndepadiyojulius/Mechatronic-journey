PROGRAM 4 · RETURN VALUE

FUNCTION
WITH RETURN
int square(int number) { int sq = number * number; return sq; } int main() { int result = square(25); printf("result = %d", result); // Output: 625 return 0; }
→ int before name = returns an integer
→ return sends value back to caller
→ square(25) → passes 25 as argument
⚡ Every sensor read function in embedded returns a value exactly like this — readTemperature() returns a float.

KEY LESSON · VOID VS RETURN
7 / 8
THE BIG
DIFFERENCE
VOID FUNCTION
Does work inside itself.
Prints, moves motors, blinks LEDs.
Returns nothing to caller.
RETURN FUNCTION
Computes and hands back a value.
readSensor() → 23.4°C
Returns a result to use.
FUNCTIONS = REUSABLE CODE BLOCKS
Every microcontroller firmware is built
from hundreds of small functions.
