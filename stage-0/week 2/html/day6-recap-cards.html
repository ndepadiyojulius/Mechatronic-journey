PROGRAM 5 · ARRAY + POINTER
5 / 8
VALUE AND
ADDRESS TOGETHER
int numbers[10] = {1,14,51,54,54, 21,36,24,25,54}; for(int i=0; i<10; ++i) { printf("%d = %p\n", numbers[i], &numbers[i]); }
WHAT YOU DISCOVERED
Each element has a value AND a memory address. Addresses increase by 4 bytes each step — exactly the size of an int. Arrays are just consecutive memory blocks.
⚡ This is how microcontrollers store sensor buffers — 10 ADC readings in consecutive memory. You just visualized it.
