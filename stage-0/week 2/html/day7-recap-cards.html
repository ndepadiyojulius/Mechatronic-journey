// Allocate at runtime int* ptr = (int*)malloc(n * sizeof(int)); if(ptr == NULL) { printf("Memory cannot be allocated"); return 0; } // Fill using pointer arithmetic scanf("%d", ptr + i); printf("%d", *(ptr + i)); free(ptr); // always free!
malloc()
allocates heap memory at runtime
free()
releases memory — always required
⚡ In embedded: malloc reserves RAM for sensor buffers at startup. FreeRTOS task stacks use this exact pattern.

PROGRAMS 2 & 3 · MACROS 🔥
4 / 8
PREPROCESSOR
MACROS
// Constant macro #define PI 3.1415 double area = PI * r * r; // Output: 483.04 ✓ // Function-like macro #define circleArea(r) (PI * r * r) double area = circleArea(12.4); // Same result — cleaner code!
WHY #DEFINE MATTERS
Macros run at compile time — zero runtime cost. Every embedded firmware uses #define for pin numbers, baud rates, buffer sizes, and hardware constants.

PROGRAM 4 · FILE I/O 🔥
5 / 8
READING
FROM A FILE
FILE* fptr; fptr = fopen("julius.txt", "r"); char content[1000]; if(fptr != NULL) { while(fgets(content,1000,fptr)){ printf("%s", content); } } else { printf("file open unsuccessful"); } fclose(fptr);
→"r" = read · "w" = write · "a" = append
→Always check ptr != NULL before reading
→Always fclose() — releases file handle
⚡ Data loggers in embedded write sensor readings to SD cards using this exact FILE* pattern.
